# Artilate 🍫

**Artilate** is an online chocolate store offering handcrafted, uniquely filled chocolates with exotic flavors such as pistachio, passion fruit, and more.

## 🌐 Live Website

[Visit Artilate Online Store](https://artilate.com)

## 📦 Features

- 🛍️ Beautiful online shop with modern design
- 🍫 Wide selection of artisanal filled chocolates
- 💳 Shopping cart and checkout system
- 📱 Responsive design (mobile, tablet, desktop)
- 🔍 Browse by flavors and categories
- 🧾 Order summary and confirmation

## ⚙️ Tech Stack
